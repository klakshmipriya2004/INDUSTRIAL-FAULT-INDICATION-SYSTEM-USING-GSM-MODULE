# Industrial Fault Indication System using GSM
LPC2148-based Industrial Fault Indication System that monitors temperature and humidity, detects fault conditions, and sends SMS alerts via GSM using UART communication. I2C EEPROM is used to store password and sensor data, with LCD display and keypad for secure user interaction.

## Hardware Used
- LPC2148 Microcontroller
- GSM Module
- Temperature and Humidity Sensor
- 16x2 LCD
- Matrix Keypad

## Communication Protocols
- UART (GSM communication)
- I2C (EEPROM for password and data storage)

## Programming
- Embedded C
